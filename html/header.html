Oodebe Utils

Oodebe is an asynchronous IO based REST API framework built on node.js.
  1. Provides facilities to develop, test and deploy a REST API server
  2. Each API call can be constructed as a script
  3. Scripts can be executed as either native node.js scripts or one of the supported plugins
  4. Plugins allow scripts to access various REST servers such as Neo4j, SOLR, and Lily using their REST interface
  5. Additional plugins can be written in the form of simple JSON configuration files
